Introduction To HTML

The web, invented in 1990 is composed largely of HTML documents. HTML stands for Hypertext Markup Language and is the language the browsers use to interpret web pages. HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. We use HTML tags to create a simple HTML document that a browser can display as a web page.

Basically HTML is made up of :-
 1) Text content
2) Markup
3) Refrences
4) Links to other pages

HTML Markup

HTML Tags

Most HTML elements consist of an opening and a closing tag with some content between them.
Its like :-
< TAG NAME (Opening) > Contents < /TAG NAME (Closing) >

For example,

< b> text < /b> - b is for bold and it makes the 'text' content bold.
< em> text < /em> - em is for emphasis and it makes the 'text' content italic.

HTML Attributes

Some elements accept additional values called attributes in their opening tags. Attributes provide additional information to the browser.
Its like :-
< TAG NAME ATTR(attribute name) = "value" > Contents < /TAG NAME >

For example,

< a> tag - The anchor tag takes an attribute called href that defines a URL to link - Creates a hyperlink to the href attribute value url and names it link.

Adding Images

< img> tag -



Its like :-
< img src="url" alt="text">

Displays the image hosted at the src attribute value "url" (this can also be a path to a local file on your computer).
If the image cannot be displayed, then the alt attribute value "text" is shown.
For more details, we can learn from internet.

Whitespace

HTML takes all text as one line, pressing enter makes no difference. So to input text in next line we use two tags specificly :-
1) < br> tag - br stands for break. The text entered after this tag comes in the next line. Moreover, it is a void tag i.e. having no closing.
2) < p> tag - p stands for paragraph. The text in this tag comes in form of paragraph. And, it is not void tag i.e. we need to close this tag.

Inline vs Block Elements

In HTML, all the spaces between some text comes in just one single space, so we put a < br> tag there which is inline.
While < p> tag is block i.e. makes a invisible box around text which is having height and length.
